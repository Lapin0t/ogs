Scoped and sorted families

In this file we bootstrap the abstract theory of contexts, variables and substitution by defining a special case of type-families: families indexed by a set of contexts C and possibly a number of times by a set of types T.

Definition Fam‚ÇÄ (T C : Type) := C -> Type .
Definition Fam‚ÇÅ (T C : Type) := C -> T -> Type .
Definition Fam‚ÇÇ (T C : Type) := C -> T -> T -> Type .

Additionally, to represent binders we use the following alternative to sorted families Fam‚ÇÅ. In this definition, the set of binders is indexed by the set of types T but not by the set of contexts C. Instead, we have a projection into C. This is important for reasons of bidirectional typing information flow. We could go with Fam‚ÇÅ instead, but this would entail quite a bit of equality dance when manipulating it.
